.. Remark: links are different from section labels, even though they are declared in a similar way

   declare a section label:
   .. _gettingStarted_GUI:
   declare a url:
   .. _gettingStarted_GUI: http://pviz.net/docs/gettingStarted.html#gettingstarted-gui   
   
   insert ref for sections
     :ref:`gettingStarted_GUI`
   insert ref for links:
     `gettingStarted_GUI`_
     
   This is because ref will go get the section label name, whereas `foo`_ will directly take what's after `:'
